# Scientist-Bin Frontend

This is the frontend of Scientist-Bin, which provides a user interface for interacting with the multi-agent system. It allows users to input their data science problems, select the desired frameworks, and monitor the training and deployment process of their models.

**Not prioritized for now, will focus on backend first.** The frontend will be built using React and TypeScript, and will communicate with the backend through RESTful APIs. It will include features such as a dashboard for monitoring the status of training and deployment, a form for submitting data science problems and selecting frameworks, and a visualization component for displaying training results and model performance metrics.
