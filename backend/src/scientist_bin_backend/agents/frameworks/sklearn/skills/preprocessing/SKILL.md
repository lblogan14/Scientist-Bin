# Preprocessing & Pipelines Skill

This skill provides data preprocessing, feature engineering, and pipeline
construction patterns for scikit-learn workflows.

## Capabilities

- Data loading (CSV, Excel, TSV, Parquet)
- Feature type identification (numeric, categorical, datetime)
- Missing value handling (SimpleImputer, KNNImputer, IterativeImputer)
- Feature scaling (StandardScaler, MinMaxScaler, RobustScaler)
- Categorical encoding (OneHotEncoder, OrdinalEncoder, TargetEncoder)
- Feature engineering (datetime extraction, polynomial features, binning)
- Pipeline construction (Pipeline, ColumnTransformer, FeatureUnion)
- Dimensionality reduction (PCA, SelectKBest, SelectFromModel)

## When to Use

Use this skill when building preprocessing pipelines. Examples:

- "How to handle mixed numeric and categorical features?"
- "Which scaler for data with outliers?"
- "How to build a ColumnTransformer pipeline?"
- "Feature engineering for datetime columns"
